# Acoustic Detection Data

Hourly acoustic detection data by fish (capture) and receiver.

## Usage

``` r
detection
```

## Format

A tbl data frame:

- DateTimeDetection:

  The detection date and hour (time).

- Capture:

  The fish code (fctr).

- Receiver:

  The receiver code (fctr).

- Detections:

  The number of detections in the hour (int).
